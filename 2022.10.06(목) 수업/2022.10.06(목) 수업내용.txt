기타 내용

툴 박스 열었다 닫기 -> T Key
툴 박스에서 바꾸는 단축키 -> W key 

캐릭터 만들때 주의할 점
  캐릭터 모델링시 Face간의 간격이 일정해야 예쁘게 나옴.
   [ Face 즉, 면 간격이 너무 클경우 찢어지는 느낌이 나버림 ]
   [ Subdivision이 아니라 Loop Cut으로 넓히니까 Mirror도 제대로 안먹고, 모양도 이상함 ] 

Edit Mode에서 상단 Mirror X,Y,Z 를 사용하면 대칭 되는 부분 조절이 가능함.

Shrinkwrap -> 선택한 Target에 달라붙게 만드는 Modifier

------------------------------------------------------------------------------------

배운 내용

1 ) 작업 전 사전 셋팅
    1. Image Referece 불러와서 정면 옆면 배치하기
    2. 해당 이미지의 하이라키뷰? 트리뷰? 마우스 커서 체크해서 선택 안되게끔 방지하기
    3. Image Properties로 이동해서 Front로 옮기고, Opacity 값으로 불투명도 설정하기.

2 ) 다람쥐 캐릭터 머리 만들기
    1. Cube 생성 후, Subdivision 2정도 주고, Apply
    2. Edit Mode의 Mirror는 좌우 대칭이 아닐경우 적용이 안됨, 따라서 Modify Properties에서 Mirror 사용
    3. 생성했던 Cube를 Vertex의 절반을 제거하고, Mirror Modify 추가하기. Cliping 체크하기
    4. Propotinal 적용하고 마우스로 볼 부분 만들기.
    5. 시점 돌려가면서 전체적으로 얼굴 대략적인 Vertex 이동해가면서 모형 맞추기

3 ) 다람쥐 귀 만들기.
    1. Top View에서 Face 2개 선택하고, Extrude 하기
    2. 전체적인 View를 돌아보면서 귀 모형으로 구현하기
    3. 가운데 Vertex를 잡고 안쪽으로 넣어주기. 귀에 둥근 느낌 표현될 수 있게

4 ) 입 만들기
    1. 적절하게 입 부분이 들어갈 위치 찾기.
    2. 얼굴을 Front View에서 보았을 때, 입을 만들려면 Vertex가 필요한데 Mirror로 작업하면 없음.
       [ 따라서, 사용하던 Mirror를 Apply 해서 양면에 Vertex 만들기 ]
    3. 적당한 입가 위치에서 Vertex 하나 잡고 V Key로 Vertex 찢기.
    4. 주변 Vertex를 선택한 상태로 G Key를 더블 클릭으로 위치 값 조절
       [ Vertex로 이동시, X,Y,Z Key로 조절하면 안됨 <- 얼굴이 평면이 아니라 곡선이므로 ]
    5. 입 주변에 Vertex 끝점들 선택하고 Extrude로 얼굴 안쪽으로 집어넣고, 다시 Extrude 크기 넓혀놓기.
    6. Subdivision 3정도로 전체적으로 부드럽게 만들기

5 )  코 만들기
    1. 입 위쪽에 중심 설정하기
    2. Cube 생성해서 적당하게 크기 조절하기
    3. Loop Cut으로 3개정도 세로로 나누기
    4. 가운데 Vertex 선택하고 위로 올려주기.
    5. 양끝 Vertex들 선택하고 크기 줄이기
    6. 나머지 Vertex들 눈으로 보면서 맞춰주기

5 ) 코 위에 작은 코 만들기
   1. Cube 생성하고
   2. Loop Cut 이후에 다이아몬드 형태로 Vertex 조절하기
   3. 코 끝에 배치하기
   4. 양끝을 회전시켜서 부드럽게 만들기.
   5. Shrinkwrap 기능 사용 큰 코 선택하기
   6. Solidify 로 두께감 넣기 Thinkness 0.17 , Offset 1
   7. Subdivision 3 정도 설정
   8. Smooth Shader 설정

6 ) 이빨 만들기
   1. Cube 생성 후 새로축으로 길게 늘린 모양 만들기
   2. Subdivision 3 및 Shader Smooth 적용학
   3. LoopCut 가운데 하나 세로 엣지 추가하기 
	-> 양 옆 엣지 쪽에 Loop Cut  추가하기 
		-> 아래쪽 엣지에 Loop Cut 추가하기
   4. 아래 쪽 Vertex 잡고 V Key로 찢기 그 후 각각의 Vertex들 사이 벌리기
   5. 찢으면서 생긴 빈 Face를 각 Vertex 4개씩 잡고 F key로 면 채워넣기
   6. 치아가 튀어나오는게 예쁘므로, 윗쪽 Vertex 잡고 안으로 집어넣거나 밖으로 빼면서 조절하기

7 ) 몸통 만들기
   1. Cube 만들고, Subdivision 2정도 주고 Apply
   2. 크기 적절하게 만든 후 Subdivision 다시 적용, Apply 하지 않기
   3. Vertex 정면에서 절반정도 선택한 후 제거하고, Mirror Midifier에서 적용하기
   4. 옆면에서 Propotinal 기능 활성화 한 상태로 몸통 크기 맞춰주기

8 ) 몸통에서 팔 만들기
   1. 한쪽 면에서 Face 잡고 Extrude를 통해서 팔 뽑기
      ( 상단 Mirror X? Y축 활성화해야, 대칭적용 됨 )
   2. Scale을 앞면에서 봤을 때, 보면서 적절하게 맞춰주기

9 ) 몸통에서 다리 만들기
   1. 한쪽 면에서 Face 잡고 Extrude를 통해서 다리 뽑기
      ( 상단 Mirror X? Y축 활성화해야, 대칭적용 됨 )
   2. Scale을 앞면에서 봤을 때, 보면서 적절하게 맞춰주기

10 ) 발 만들기
   1. Cube 생성하고, 발 크기에 맞춰서 크기 조절
   2. Loop Cut으로 발가락 개수만큼 면 나누기
   3. Extrude로 각각의 면을 발가락처럼 앞으로 뽑기
   4. SubDivision으로 부드럽게 만들기
   5. 각각의 Vertex 위치 조절해가면서 각진 부분 없얘기
   6. Shader Smooth 적용하기
   7. 발이 너무 평평하니까 발목을 만들어줘야되므로, 만들고 싶은 부분에 Loop Cut 적용하기
   8. 적용한 부분의 Vertex 2개 잡고 위로 올려서 발목 만들어주기
   9. 발등이 너무 평평하니까 둥글게 만들기 위해서 앞쪽 발 Vertex 잡고 둥글게 만들어주기
   10. Mirror를 사용해서 양쪽 발을 렌더링해주기.

11 ) 손 만들기 
   1 ~ 10. 발만들기와 원리가 동일함.
   [ 단, 손가락 줄일때, 각각의 Vertex를 줄일거면 . key Median Point가 아닌, Individual Origins로 설정하고 스케일 조절할 것. ]
   [ 엄지손가락은 Loop Cut -> Extrude로 만듬. 동일함 ]
