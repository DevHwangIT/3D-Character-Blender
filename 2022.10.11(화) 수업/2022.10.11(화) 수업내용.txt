기타 내용

Texture Paint 내부 설명

사이즈는 F key 세기는 Shift + F key

- Stork에서 Brush 종류가 많음. 
Line 사용할 때 -> 다람쥐 직선 무늬 만들때와 같이 선으로 된 것을 그을 때 사용,
Curve 사용할 때 -> 포토샵으로 치면 Pen Tool과 같음.
[ Ctrl + 오른쪽 마우스로 점 새로 찍기, 그냥 잡고 움직이는 것은 오른쪽 마우스 버튼 ]
[ 색칠은 Ctrl + 왼쪽 마우스 클릭하기 -> 클릭하면 기존에 생성한 라인에 따라서 칠해짐 ]

- FallOff에 관한 설명
아래의 그래프 모양에 따라 실제 Brush로 칠했을 때, 경계선의 진함이 다름.

- Symmetry에 관한 설명
Mirror와 유사하게 X,Y,Z 축 복사가 되는 기능임.
Texture Paint에서 지우개 기능은 없음. 지우려면 바탕화면을 스포이드로 찍어서 지워야 됨.
Texture Paint의 Image가 저장안되어 있을 경우, Image 메뉴에 *표시가 뜸.
[ 텍스처 작업하기전에 이미지를 저장하기 필수임. ]

- 기타 작업 팁 
Texture Paint 작업시 렉이 걸리면 Subdivision의 레벨을 낮춰서 작업하면 빨라짐.
[ Vertex에다 칠하기 때문에 ]
텍스쳐 작업시에 중간중간 꼭 저장할 것. 텍스쳐 작업은 자주 튕김.
텍스쳐 칠할 때, 색상이 잘 안보일 수도 있음. ViewPort Shading의 설정에서 Lighting을 Flat으로 변경

- Texture Paint 좌측 탭 View
Fill( 포토샵 페인트 아이콘) 한가지 색상으로 칠하는것
Soften( 물방울 아이콘 )은 경계선을 부드럽게 만들어주는 기능

배운 내용

1 ) Point Light 눈동자에 빛나는거 없얘는 방법
   1. Multiple ~? 를 체크해서 없애주면 사라짐.

2 ) 눈썹 만들기
   1. Plane 생성후 Edit Mode에서 뒤쪽 Vertex 제거하고, 선분으로 만들기
   2. Snap Face로 눈가 주변에 선 붙이기
   3. 끝 Vertex하나 잡고 Extrude로 선분을 이어서 눈의 절반까지 이어붙이기
   4. Side View로 이동한 후, Snap 기능 비활성화로 변경하고 Extrude로 눈썹 세우기
   5. Mirror로 얼굴복사해서 양쪽 추가하기.
   6. Solidify로 두께감 주고, Subdivision으로 부드럽게 만든 후, Smooth Shader 적용.
   7. 눈썹 뒤쪽 Extrude로 피부에 파고들게 만들기.
   8. 눈 앞쪽 Vertex 선택 후 가깝게 만들어서 눈썹 얇게 만들기
   9. 눈꼬리 빼는법은 Edit Mode의 각각 2개의 Vertex 끝 쪽 잡고, Extrude 하고나서 Vertex 4개 잡고 F key(Fill)로 채우기
   10. 9번의 과정에서 생겨난 Vertex와 면을 적절하게 이동으로 늘려서 눈꼬리 만들기.

3 ) 입 안 만들기
   1. Cube 생성해서 Face 앞부분 제거하기
   2. Subdivision 2정도로 부드럽게 만들기
   3. Shader Smooth 적용해서 더 부드럽게 만들기
   4. 위치 잘 배치하기

4 ) 꼬리 만들기
   1. 꼬리 쪽에 Cube 생성해서 한 쪽방향으로 길게 늘려주기
   2. Xray 확인하면서 뒤쪽 Vertex들 잡아서 Extrude 적용하기 ( Extrude 중에 Z Key누르면 방향이동 가능 )
   3. 꼬리 느낌으로 반복해서 늘리기
   4. 전체적인 Vertex 조절해가면서 꼬리 느낌 살리기
   5. 중간에 긴 면은 LoopCut으로 여러개 엣지 만들고 Vertex 조절하면서 늘리기
   6. 꼬리 형태로 다듬기.

5 ) 얼굴 텍스쳐 만들기
   1.  Shading 창으로 이동 후 Material 생성하기
   2. Image Texture Node 생성 후 New 버튼으로 텍스쳐 만들기 4096 정도 사이즈로 만들기
   3. 텍스쳐 칠하기전에 앞서서 UV를 펴야함.
   4. 일반적으로 캐릭터나 얼굴들은 Side View로 이동해서 가운데 수직선을 기준으로 Mark Seam 적용함.
   5. 모든 Vertex A key로 선택 후 Unwrap 적용하기
   6. 앞 부분만 칠할 예정이므로, 앞 부분 UV만 Pack Island로 키우고 뒷부분 UV는 구석으로 작게 줄여놓기
   7. Texture Paint 창에서 N - Tool 이동해서 Brush 설정하기
   8. Texture Paint Brush 등 설명 진행하기

6 ) 몸통 텍스쳐 만들기
   1. 5번의 과정과 동일

7 ) 손 텍스쳐 만들기
   1. 5번의 과정과 동일
