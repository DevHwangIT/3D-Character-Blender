배운 내용

1 ) 바닥 면 채워서 몸통 만들기
  1. 바닥 Edge 잡고, Poly Build로 선 늘리기
  2. LoopCut이나 및 이동 스케일로 균일하게 작업하기.

2 ) 눈 내부도 채우기
   1. 눈 내부도 채우기
   2. 눈 외곽 LoopCut으로 경계선 늘려주기

3 ) 와이어 프레임 확인하기
   1. Mirror로 Cliping 된 부분 접합 이상한지 확인하고 수정하기
   2. Mirror Apply 하기전에 UV 먼저 피고 Apply 적용하기
      [ 적용한 후, UV 피면 작업량이 2배임. 따라서 미리 하고 Apply 처리 하는것임. ]
   3. 머리, 목, 몸통으로 Alt 선택 후 MarkSeam 심기.
   4. 옆면에서 가운데 Edge Alt 선택하기. 그 후에 귀는 Ctrl 누른 상태로 선택 취소하고,
      원형으로 감싸서 선택하기 마지막으로 윗부분 머리중앙도 선택해서 반으로 가르기
   5. 눈 흰자도 감싸고, 입술도 감싸고, 코도 크게 감싸서 심을 심는다.
   6. 머리 뒷면은 UV펴도 생각보다 중요하지 않을 것 같으니까, 조금살짝 앞으로 나와서 마크심 다시 심었음.
   7. 귀 앞부분도 떼어내주기 위해서 클릭해서 기존 마크심과 이어진 것 확인하고, 마크심 심기
   8. Mirror Apply 적용
      [ Mirror Apply 적용하면 뒤로가기 어려우므로, Save As로 다른 파일로 저장하고 작업하기 ]
   9. 리토폴로지 한 Plane 오브젝트에 Shrinkwrap 속성 스컬팅 오브젝트 선택 후 달라붙게  만들기
   10. 쉬링크적용한 이후에 MultiResolution 5 Level 정도 적용하고, 위로 올리기.

4 ) 전체적인 모델 디테일 수정 작업
   1. Sculpt Mode 혹은 Edit Mode로 이동 했을 때, 찢어지는게 안보일 때는 Shrinkwrap Apply 후 이동
      [ Vertex가 많아지면서 발생하는 문제임. 또, 이제 필요없는 Sculpt 작업한 오브젝트는 삭제함.
   2. Sculpt Mode에서 펜으로 전체 돌아보면서 부드럽지 않거나,찢어진 부분에 대해서 Shift key로 Smooth 칠하기
      [ 작업할 때, Mirror 키고 펜으로 작업하기. ]
   3. 눈이 렌더링시에 안나오는 건, Object Properties 에서 In - Front 체크 확인하기.
   4. Sculpt Mode에서  Crease(파란색)로 눈가 주름 만들어주고 Draw Ctrl 누른 채로 눈 주름 판후, 부드럽게 만들기.
   5. Draw Sharp Brush(파란색)로 점막을 그리기
   6. 점막 부분 크기는 Elastin Deform Brush(노란색)로 크기 맞춰주기.
      [ 할 내용이 많아서, 대략적인 방법만 설명한 후에, 다른 부분으로 이어 진행할 예정임 ]
   7. 콧볼 및 인중 Draw Brush(파란색)로 파기.
   8. 웃으면 광대가 나오고, 입꼬리가 들어가야됨. 이를 위해서 입꼬리를 Elastinc Deform으로 당기고, 앞광대는 살짝 당긴다.
   9. 입술모양 바꾸고 싶으면 똑같이 Deform으로 잡아서 수정해주기.
   10. Crease ( 파란색 )으로 귀 내부 모양 파준 그대로 따라서 그리면서 각 살려주기.
   11. 귀 내부를 Draw Brush ( 파란색 )으로 볼록한 느낌을 만들고 Crease (파란색) [ Shift+ C key ] 로 귓 파인 부분 및 귓구멍을 만들어준다.
        [ 리토폴로지 과정에서 귀를 잡으면 좋긴하나, 그러면 너무 어려워지기때문에, Multiresolution 과정에서 귀 안쪽 내부를 처리한다 ]
        [ Front View에서 봤을 때, 귀 내부가 튀어나온 느낌이 곡선으로 아름답게 나올 수 있도록 리터칭 해준다. ]
   12. 목뼈를 위해서 Draw로 직선이 아닌, 실제 목선처럼 곡선 라인따라서 그리기
   13. 목뼈에서 파인 부분을 Crease로 파인 연출하기
   14. 전체적으로 Smooth 및 Elastin Deform으로 조절하기
   15. 목뼈 끝나는 부분에 다시 Draw Brush로 ㄱ자로 만들어 주고 디테일 수정하기.
   16. 목뼈 만든 부분끼리 연결되는 곳 디테일하게 수정하기
   17. UV 부위별로 파악해서 작업이 중요한 부분은 크게크게 그려서 펴기
        [ UV 공간 내부에서 UV의 크기는 해상도를 나타낸다. ]
   18. 얼굴의 중요한 부분은 가운데 얼굴 모형인데 목이나 이마등 쓸데없는 부분이 큼. 이를 해결하기 위한 방법
        [ Vertex  선택모드로 이동해서 중앙부분찍은 후 Propotinal 활성화 상태로 Scale 키우기]

다음 수업은 Texture Paint와 시간이 되면, 머리카락과 속눈썹을 만들예정임.
